About the Boilerplate

This boilerplate is a pre-configured setup for the Angular coding round at Tarkashilpa. It includes:

  • ✔ SCSS as the CSS pre-processor
  • ✔ Phosphor as the icon library. Browse available icons here.

Test Instructions

  • ✔ The test must be completed with screen sharing enabled.
  • ✔ Candidates may refer to Angular.dev for syntax and Phosphor Icons for icons during the test.
  • ❌ Using any other websites, blogs, Stack Overflow, Google, ChatGPT, or AI-generated code is strictly prohibited.
